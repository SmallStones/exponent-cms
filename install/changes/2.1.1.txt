version 2.1.1
This release focuses on the new forms module and e-Commerce as OOTB experience
  - adds many new event calendar features
    -- adds event copying feature (create new event from existing event)
    -- adds optional images to events and event registrations
    -- adds optional event popup in a lightbox dialog (like google calendar)
    -- adds ajax-based navigation in calendar months (weeks/days), news items & blog posts to prevent reloading entire page
    -- adds 'cancel' event feature to display, yet mark events as cancelled
  - adds grouping by date for uncategorized portfolio and file download items (in addition to grouping by alpha for rolodex feature)
  - adds custom module phrase translation libraries feature
  - adds a mass mailer for super admins to email all or selected site users
